How to Use

  1. Make sure Streamer.bot is running with the WebSocket server enabled (default: 127.0.0.1:8080)
  2. Open OBS and add a Browser Source
  3. Enter the overlay URL (e.g., {{ site.url }}{{ site.baseurl }}/overlays/sample-timer)
  4. If your Streamer.bot uses a different address/port/password, add them as URL parameters: ?address=127.0.0.1&port=8080&password=yourpassword
  5. Set the Browser Source width to 1920 and height to 1080 (or your stream resolution)

How It Works

These overlays are standalone HTML files that connect to Streamer.bot's WebSocket server. When you add them as Browser Sources in OBS, they establish a WebSocket connection to your local Streamer.bot instance and listen for events. The overlays can display timers, alerts, notifications, and more based on Streamer.bot actions and events.

Each overlay is self-contained and uses the @streamerbot/client JavaScript library to communicate with Streamer.bot. You can customize them by modifying the HTML files or adding URL parameters for configuration.
